## Módulo 04:

-- Caso queira usar o projeto, ele contém as depedências:
   * Nunjucks
   * Browser-sync
   * Express
   * Nodemon
   * Npm-run-all

Para instalar todas as depedências é só usar o comando: 
```sh
npm install
```
